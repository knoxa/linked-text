# WW1 Events

## Input

* [Principal Events 1914-1918](https://tigersmuseum.github.io/history/events/ww1/events-1914-15.xhtml)

## Results

* [Military Operations](results/operations.xhtml) - Start and end dates of events describing military operations.
* [Entity name map](results/entity-name-map.xml) - A lookup table where the key is a name as it appears in surface text (an alternate name), 
and the value is a normalized version (preferred name).
* [Entity type map](results/entity-type-map.xml) - A lookup table where the key is a preferred name and the value is an entity type.
